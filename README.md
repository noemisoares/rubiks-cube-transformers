# rubiks-cube-transformers
Criar uma simulação interativa de um cubo mágico (Rubik’s Cube) em Three.js, com câmera 3D e animação suave ao girar as faces.

## Temas
- Autobots: tema padrão com a paleta laranja, azul e amarela do universo Transformers.
- Decepticons: tema escuro com vermelho, roxo e contraste agressivo.
- Energon: tema vibrante com ciano, amarelo e brilho energético.

## Referências
- Fontes: Google Fonts — Bebas Neue, Orbitron e Rajdhani.
- Ícone local: icons/autobots-icon.png.
- Estética inspirada no universo Transformers.
